## Module Responsibility Map

| Module | Responsibility |
|--------|---------------|
| `config_parser.py` | Parse `/*FACTORY*/` block from SQL; extract title, theme, DB type, slot assignments |
| `mquery_generator.py` | Generate Power Query M code for all supported source types (CSV, Excel, Oracle, PostgreSQL, Snowflake) |
| `factory.py` | Orchestrate the full pipeline; copy template; patch file names, displayNames, M Query; call all sub-pipelines |
| `visibility_pipeline.py` | Mark unused template slots as hidden in TMDL (`isHidden: true`, `IsAvailableInMDX: false`) |
| `format_pipeline.py` | Apply `formatString` metadata to active columns and measures in TMDL |
| `rename_pipeline.py` | Rename all field references across TMDL, DAX, relationships, Field Parameter tables, and visual JSON files |
| `sort_pipeline.py` | Wire `sortByColumn` for ORDER columns; hide ORDER columns |

## Key Design Decisions

### lineageTag is Inviolable
Power BI uses GUIDs (`lineageTag`) to track field identity across visual bindings. The rename pipeline changes `name`, `sourceColumn`, and all string references вҖ” never the `lineageTag`. This is why renames do not break visual bindings.

### Unused Slot Strategy
Unused slots are eliminated at two levels:
1. **M Query** вҖ” `Table.RemoveColumns` drops them entirely from the data pipeline
2. **TMDL** вҖ” `isHidden: true` + `IsAvailableInMDX: false` hides them from visuals and metadata

Visuals never see empty columns; the Details table never shows null-only fields.

### DAX Measures vs. M Query Columns
`DAX_SUM_Measure_*` and `DAX_CNT_Measure_*` are semantic model calculated measures (not CSV columns). They are excluded from `Table.RenameColumns` in the M Query but renamed alongside their corresponding column slots in TMDL and Field Parameter tables.

### Field Parameters
Three Field Parameter tables enable dynamic chart switching without custom DAX:
- `Select Dimension` вҖ” switches chart axis (Key_Dim slots)
- `Select 2nd Dimension` вҖ” secondary axis switcher
- `Select Measure` вҖ” switches chart values (DAX measure slots)
